Portfolio

Welcome to my portolio! I am a developer currently based in the San Francisco Bay Area of California. Here you can find some of my favorite personal, and professional, projects.

Projects

  • Luminary Bay Farms

    This small business website for microgreens was my first professional project built from scratch. The website showcases my ability to build a project from the ground up, using React, to create a single-page application with plenty of content. Using JavaScript, I created data tables and functions to be accessed throughout the site, and each of the route elements adhere to the responsive CSS I designed.

    Although I was provided with all the images used, I was given the freedom to make all of the decisions for the design and structure. I chose the colors, layout, and designed their logo. I also wrote all of the content for the business owner on every page. Both I and the business owner are very excited about how it turned out!

    Check it out!

  • Reddit-Zero

    This project is a recreation of Reddit, using Reddit's JSON API, for simple viewing of posts and comments from popular categories. I built this application using a course project as the foundation. Admittedly, this project was easily the most difficult build I have accomplished so far, but I also found it to be the most rewarding.

    The project started as a way to enhance my skills with JavaScript, but I came to see it's value as a potfolio piece very quickly. Throughout all my troubleshooting, I learned a great deal more about APIs than I had in all of my courses, and I even picked up a few CSS tricks along the way!

    Click Me!

  • Altannertive

    This final project is another API application, based on Spotify's music library, that I built during my Front-End Development course. The function of this application is to allow you to create playlists on your Spotify account by searching song titles, albums, or artists. After the list populates, you can add or remove any item to a 'New Playlist' with a title of your choice.

    After fulfilling all of the parameters for the project in the course, I added a full design and made it my own, inspired by a playlist of my own that I had dubbed Altannertive. Although the application is quite simple, It was a very enjoyable start to my career.

    Let's Talk Music!
